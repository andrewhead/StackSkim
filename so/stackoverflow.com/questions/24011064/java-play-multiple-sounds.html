Java: play multiple sounds

up vote 0 down vote favorite

I'm looking for a way to play sample sounds (less than 1sec), but I want to play them at the same time.

On this site, I've found several examples of code like this one, directly copied from another response:

sound = AudioSystem.getClip();
//read audio data from whatever source (file/classloader/etc.)
audioSrc = getClass().getResourceAsStream("myResource.wav");
buffer for mark/reset support
bufferedIn = new BufferedInputStream(audioSrc);
audioStream = AudioSystem.getAudioInputStream(bufferedIn); // java.io.IOException: Stream closed
sound.open(audioStream);

But I catch java.io.IOException: Stream closed. I've two questions:

  • Why is the stream closed?
  • Is this the best way to play sounds? Another library can't do better? (i.e. simplier)
