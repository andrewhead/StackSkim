    Retrieve the current URL hash. This method is also mostly for supporting automated testing, but also does some basic error handling for malformed URLs.

      _getHash() {
    try {
      return decodeURIComponent(location.hash.slice(1));
    }
    catch(e) {
      return e;
    }
  }
