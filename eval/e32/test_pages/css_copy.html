To highlight something on the page with the selector, type in:

$("selector").css({'background-color': 'yellow'});

where you change the selector and leave the rest of the line the same.

Open up the CSS test bookmark, then type Ctrl+Alt+K, and then paste in the line above. Modify the selector to choose a specific element.

Here's an example we'll both try.

$("pre").css({'background-color': 'yellow'});
