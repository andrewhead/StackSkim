Instructions for Code Modification

The purpose of this study is to see how you modify existing code to perform new tasks. In the following examples, I will ask you to locate relevant fragments of code in snippets I provide, and use them to perform a goal that I specify.

More concretely, for each task you will:

  1. Read and understand a programming goal.
  2. View a code snippet and, sometimes, an accelerator.
  3. Write a short piece of code that completes the goal using what you found in the code snippet.

After viewing the snippet and accelerator, please use whatever tools that will most productively help you understand and solve the problem, including the Internet.
