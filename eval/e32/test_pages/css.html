An Introduction to Accelerators

Accelerators are context-relevant descriptions of code. You will use them to read descriptions of CSS selectors.

For example, here is a CSS selector in this line of code:

$("img");

You can view an accelerator for code by highlighting the relevant code. In this case, select the text between the quotations to view a description of the img selector.


Accelerators work anywhere on the web. The proctor will show you an example here.
